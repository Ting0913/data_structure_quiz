# 資料結構時間複雜度線上測驗系統 💡

這是一個基於 Django 框架開發的「時間複雜度分析」線上測驗與智能分析網站，旨在協助學生透過互動式練習掌握演算法漸近分析。

## 🚀 核心功能與系統亮點
* **會員帳號系統**：支援註冊與登入，並能永久儲存個人作答歷史紀錄。
* **動態錯題複習機制**：系統會自動透過 Session 記錄使用者的錯題，並在測驗中優先抽題複習。
* **即時排行榜**：採用嚴謹的排序邏輯，優先依據「得分率高」排序，若得分率相同則依據「花費時間少」進行排名。
* **觀念解析與數學公式美化**：交卷後提供微亮黃色的優雅詳解卡片，並成功整合 MathJax CDN，能完美在網頁上渲染出漂亮的 Big-O 數學符號（如 $O(n \log n)$）。
* **個人作答記錄與歷程追蹤**：使用者登入後，系統會自動從資料庫撈取該帳號的歷史作答數據，完整呈現作答時間戳記（Timestamp）、測驗得分率以及花費時間，讓大家能追蹤自身的學習進度。

## 🛠️ 技術棧與開發環境
* **後端 (Backend)**：Python 3.x / Django 框架 / SQLite 資料庫
* **前端 (Frontend)**：HTML5, CSS3, JavaScript, MathJax
* **開發工具 (IDE)**：Visual Studio Code
